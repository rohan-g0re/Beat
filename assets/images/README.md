# Images

Place your app images and illustrations here.

## Required Assets

### App Icon
- `app-icon.png` - 1024x1024px PNG (no transparency)
- Used for iOS and Android app icon
- Expo will generate all required sizes

### Splash Screen
- `splash.png` - 1284x2778px PNG (iPhone 13 Pro Max size)
- Centered logo/branding on solid background
- Expo will handle safe areas and resizing

### Android Adaptive Icon
- `adaptive-icon.png` - 1024x1024px PNG
- Centered foreground element (use 66% of canvas)
- Transparent background

## Optional Assets

### Illustrations
- `illustrations/empty-routines.png` - Empty state for routines list
- `illustrations/onboarding-welcome.png` - Welcome screen illustration
- Recommended size: 2x resolution for retina displays

## Design Guidelines

- Use PNG format for raster images
- Include @2x and @3x versions for retina displays (if not using SVG)
- Keep file sizes optimized (use tools like TinyPNG)
- Maintain consistent visual style across all illustrations

## Asset Generation Tools

- [Figma](https://www.figma.com/) - Export designs
- [Asset Resizer](https://resizeappicon.com/) - Generate all icon sizes
- [TinyPNG](https://tinypng.com/) - Optimize PNG files
